# DNS

Domain Name System (DNS) is a method of translating IP addresses that servers and routers recognize into user-friendly, given names (i.e. softlayer.com). While the concept of DNS is fairly simple, managing and storing records for your various domains can become quite tedious if there isnâ€™t a convenient method to do so. Softlayer DNS provides customers a central location to view and manage their domains through our basic DNS management interface and also gives users the option to manage reverse and secondary DNS in the same location free of charge.
