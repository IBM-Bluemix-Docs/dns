# 檢查網域的傳播狀態

[客戶入口網站 ![外部鏈結圖示](../../icons/launch-glyph.svg "外部鏈結圖示")](https://control.softlayer.com/) 上的**檢查 DNS**工具可讓您檢查最新的時間項目已傳播到 DNS 伺服器，以及所選網域及其主名稱伺服器的標準傳播時間。請遵循下列步驟來檢查「客戶入口網站」上的網域 DNS 傳播狀態：

1. 在**網域**文字框中輸入網域。

2. 按一下**檢查 DNS** 按鈕。結果將出現在畫面右邊。
