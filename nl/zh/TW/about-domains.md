# 關於網域
{:#about-domains}

IBM Cloud 利用[客戶入口網站 ![外部鏈結圖示](../../icons/launch-glyph.svg "外部鏈結圖示")](https://{DomainName}/) 提供網域管理功能給所有客戶。不需要向協力廠商登錄、續約及管理您的網域。我們的客戶可以取得網域、續約及傳送網域，並且可以使用相同的工具立即管理其網域。
